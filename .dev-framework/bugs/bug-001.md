# BUG-001: Duplicate placeholder-panel shape across three plugins

**Severity**: low
**Status**: open
**Created**: 2026-07-04T13:46:56Z

## Description
`InvestmentsPlugin.tsx`, `HomeAutomationPlugin.tsx`, and the pre-existing `RpiDesktopPlugin.tsx`
each define an identical centered placeholder-panel style object and JSX shape. Flagged by the
Simplify agent during the `additional-features` reviewer pass.

## Steps to Reproduce
Compare `packages/frontend/src/plugins/investments/InvestmentsPlugin.tsx`,
`packages/frontend/src/plugins/home-automation/HomeAutomationPlugin.tsx`, and
`packages/frontend/src/plugins/rpi-desktop/RpiDesktopPlugin.tsx`.

## Expected Behavior
N/A — not a functional defect, purely a maintainability note.

## Actual Behavior
Three near-identical inline style/JSX blocks. The Architect explicitly deferred extracting a
shared `PlaceholderPlugin(name)` helper for the `additional-features` workspace (only 3 instances,
premature abstraction). Revisit if a 4th placeholder is ever added.

## Root Cause
Deliberate deferral, not an oversight — see `artifacts/additional-features.architect.md`, Tech
Decisions.

## Fix
Not scheduled. Extract a shared helper only once duplication grows further.
