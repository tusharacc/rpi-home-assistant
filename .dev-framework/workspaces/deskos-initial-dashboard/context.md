## Overview

Building the initial DeskOS dashboard interface for a Raspberry Pi-based personal information appliance. This workspace focuses on creating the foundational UI with sidebar navigation and content area, designed to work on macOS and deploy unchanged to Raspberry Pi.

## Key Notes

- Will be tested on macOS, deployed to Raspberry Pi OS
- Using frontend-design skill for UI implementation
- Based on DeskOS_Requirements_v0.1.md specification
- Plugin-based architecture for future extensibility
